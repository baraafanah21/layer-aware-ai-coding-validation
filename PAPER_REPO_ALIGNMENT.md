# Paper-Repository Alignment Notes

No manuscript source file was present in this repository during the final packaging pass, so the paper itself was not edited directly. The following text can be inserted into the manuscript without changing the study argument.

## Companion Repository Sentence

> A companion repository is provided as a reconstructed validation package for the reported data and analysis outputs.

## Reproducibility Scope Paragraph

> The companion repository does not claim full raw-data reproduction of all 45 experimental sessions. The original session-level files were not fully available after project-file damage, so the repository preserves the author-provided study values and recomputes only the parts that can be reconstructed from the recovered inputs. Values that depend on missing session-level rows, unrecovered rubric mappings, preserved correlation matrices, or the unrecovered TDI formula are explicitly labeled in the validation outputs. No measurements are fabricated or adjusted to force agreement.

## Methodology / Limitations Wording

> The results should be interpreted together with the reproducibility boundary of the companion package. Task-level values that were recovered are preserved and partially recomputed, while metrics requiring unavailable raw runs or supplementary mappings are reported as preserved source values. This does not change the main comparison between Manual, Full-AI, and Hybrid approaches, but it limits the extent to which every aggregate can be independently reconstructed from raw session data.

## Suggested Repository Footnote

> Repository status: reconstructed validation package; not a full raw-data reproduction package.
